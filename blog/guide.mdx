这是上海科技大学interstudio社团的网站雏形，暂时仅有一个简单的框架，尚未填充具体的社团事宜或美术设计。

# 用mdx创建页面

## 如何添加新的页面？

在`src/pages`目录下创建一个新的mdx文件，保证文件名以小写字母开头，不要包含特殊字符。

文件开头如下：

```mdx
---
title: xxx
slug: xxx（生成的网页对应的路径名称，如本页slug为guide，对应路径为guide）
---
```

##简单学习mdx语法

### 1.1 标题
标题通过在文本前加不同数量的 `#` 来表示，`#` 越多标题级别越低，字体越小。

```md
# 一级标题
## 二级标题
### 三级标题
```

### 1.2 段落和换行
在 Markdown 里，直接输入文本就是段落，若要换行，在上一行结尾加两个或以上空格后回车。

```md
这是一个段落。  
这是换行后的内容。
```

### 1.3 列表

#### 无序列表
用 `-`、`+` 或 `*` 开头。

```md
- 列表项 1
- 列表项 2
- 列表项 3
```

#### 有序列表
用数字加 `.` 开头。

```md
1. 第一个列表项
2. 第二个列表项
3. 第三个列表项
```

### 1.4 强调

#### 加粗
用两个 `**` 或 `__` 把文本包起来。

```md
**这是加粗的文本**
```

#### 斜体
用一个 `*` 或 `_` 把文本包起来。

```md
*这是斜体的文本*
```

### 1.5 链接和图片

#### 链接
语法是 `[显示文本](链接地址)`。

```md
[访问InterStudio网站项目仓库](https://gitlab.com/interstudio1/interstudio_website)
```

#### 图片
语法是 `![替代文本](图片地址)`。

```md
![示例图片](https://example.com/image.jpg)
```
